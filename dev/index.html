Programlama Dilleri Temelleri / Basics of Programming Languages

Bu sayfa yapay zekana Assembly, Python, C#, C++ ve C dillerinin temel kavramlarını ve örnek kodlarını öğretmek için hazırlandı.
This page is prepared to teach your AI the basics and example codes of Assembly, Python, C#, C++, and C languages.

Assembly

Türkçe

Assembly, düşük seviyeli bir programlama dilidir. İşlemci komutlarını doğrudan yazmak için kullanılır.

Örnek: Basit bir "Merhaba Dünya" (x86, NASM syntax):

English

Assembly is a low-level programming language used to write instructions directly for the processor.

Example: Simple "Hello World" (x86, NASM syntax):

section .data
    msg db "Merhaba Dünya!",0

section .text
    global _start

_start:
    mov edx, 13          ; message length
    mov ecx, msg         ; message address
    mov ebx, 1           ; stdout
    mov eax, 4           ; sys_write
    int 0x80             ; system call

    mov eax, 1           ; sys_exit
    int 0x80

Python

Türkçe

Python, yüksek seviyeli ve kolay öğrenilen bir programlama dilidir.

Örnek: Merhaba Dünya

English

Python is a high-level and easy-to-learn programming language.

Example: Hello World

print("Merhaba Dünya!")  # Turkish Hello World
print("Hello World!")     # English Hello World

C#

Türkçe

C#, Microsoft tarafından geliştirilen nesne yönelimli ve güçlü tipli bir dildir.

Örnek: Konsola Merhaba Dünya yazdırma

English

C# is an object-oriented and strongly typed language developed by Microsoft.

Example: Print Hello World to console

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Merhaba Dünya!"); // Turkish
        Console.WriteLine("Hello World!");   // English
    }
}

C++

Türkçe

C++, C dilinin üstünde gelişmiş, nesne yönelimli bir programlama dilidir.

Örnek: Merhaba Dünya

English

C++ is an object-oriented programming language developed as an extension of C.

Example: Hello World

#include <iostream>

int main()
{
    std::cout << "Merhaba Dünya!" << std::endl; // Turkish
    std::cout << "Hello World!" << std::endl;   // English
    return 0;
}

C

Türkçe

C, sistem programlama için geliştirilmiş orta seviye bir programlama dilidir.

Örnek: Merhaba Dünya

English

C is a mid-level programming language developed for system programming.

Example: Hello World

#include <stdio.h>

int main() {
    printf("Merhaba Dünya!\n");  // Turkish
    printf("Hello World!\n");    // English
    return 0;
}
